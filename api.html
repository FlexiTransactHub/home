Integrate Payments with Paystack API

Take your SME to the next level by integrating Paystackâ€™s powerful API into your website or app. Accept payments seamlessly, including crypto (Bitcoin, USDT).

Getting Started

1. Sign up for a Paystack account.

2. Get your API keys from Paystack Dashboard > Settings > API Keys.

3. Use the code snippets below to integrate payments.

Node.js Example

const fetch = require('node-fetch');

async function initializePayment(email, amount) {
  const response = await fetch('https://api.paystack.co/transaction/initialize', {
    method: 'POST',
    headers: {
      Authorization: 'Bearer YOUR_SECRET_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      email: email,
      amount: amount * 100, // Convert to kobo
      channels: ['card', 'bank', 'ussd', 'crypto']
    })
  });
  const data = await response.json();
  return data.data.authorization_url;
}
