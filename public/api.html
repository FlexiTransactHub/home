FlexiTransact API Integration

Enhance your SME with our API to create dynamic payment links for Naira and crypto transactions. Integrate seamlessly with your website or app using the FlexiTransact API.

Example API Endpoint

Below is an example of a payment link you can generate and customize via our API:

https://flexitransact.com.ng/pay.html?user=sme9452&amount=5000&product=cap&key=pk_test_2db2746da63982f62b2b1ed9283c66bd8a187029&crypto=true&naira=true&business=Airge

How to Use the FlexiTransact API

Our API allows you to create payment links programmatically by constructing a URL with query parameters. Follow these steps to integrate:

  1. Base URL: Start with the base endpoint https://flexitransact.com.ng/pay.html.
  2. Required Parameters:
    • user: A unique user ID (e.g., sme9452).
    • amount: The payment amount in Naira (e.g., 5000).
    • product: The name of the product or service (e.g., cap).
  3. Optional Parameters:
    • key: Your Paystack public key (e.g., pk_test_2db2746da63982f62b2b1ed9283c66bd8a187029), required if Naira payments are enabled.
    • crypto: Set to true or false to enable/disable crypto payments.
    • naira: Set to true or false to enable/disable Naira payments via Paystack.
    • business: Your business name (e.g., Airge).
    • accountNumber and bankName: Required if crypto=true for crypto payment details.
  4. Implementation Example: Use the following JavaScript code to generate a payment link dynamically: 
    const createPaymentLink = (userId, amount, product, paystackKey, cryptoEnabled, nairaEnabled, businessName) => {
  const baseUrl = 'https://flexitransact.com.ng/pay.html';
  const params = new URLSearchParams({
    user: userId,
    amount: amount,
    product: encodeURIComponent(product),
    key: paystackKey,
    crypto: cryptoEnabled,
    naira: nairaEnabled,
    business: encodeURIComponent(businessName)
  }).toString();
  return `${baseUrl}?${params}`;
};

// Example usage
const link = createPaymentLink('sme9452', '5000', 'cap', 'pk_test_2db2746da63982f62b2b1ed9283c66bd8a187029', 'true', 'true', 'Airge');
console.log(link);
  5. Notes:
    • Ensure your Paystack public key is valid and matches the pk_(test|live)_[a-zA-Z0-9]{20,45} pattern.
    • For crypto payments, include accountNumber and bankName in the query string if needed by your payment flow.
    • Test the link locally or deploy to a live environment to verify functionality.
